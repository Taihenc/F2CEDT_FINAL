## GUIDE

### To start front-end server

```shell
cd .\frontend\
npm install
npm start #or npm run dev
```

### To start back-end server

```shell
cd .\backend\
npm install
npm start #or npm run dev
```

### We also need MongoDB

```shell
cd .\backend\
docker-compose up
```
